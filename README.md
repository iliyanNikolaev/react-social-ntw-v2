# Тук може да се придобие представа как ще изглежда user interface-a на приложението: [LIVE DEMO HERE](https://glowing-gumdrop-34370b.netlify.app/). В момента всичко е хардкоднато, в бъдеще ще се използва [Supabase](https://supabase.com/) за бекенд и ще се добави реактивност.

# Това ще бъде финалния проект за завършване на курса React September 2026 от [SoftUni](https://softuni.bg/). Инициализиран е чрез [Vite](https://vite.dev/) (без TypeScript), като към момента единствената допълнително интегрирана библиотека е react-router-dom.
